# Scholarships-for-minorities

A Java console application that helps students discover scholarships based on their minority group, school status, and GPA.

## ✨ Features

- 🎓 **Scholarship Suggestions**: Recommends scholarships based on GPA
- 👤 **User Input System**: Collects student information through a menu
- 📊 **GPA Validation**: Ensures GPA is between 0.00 and 4.00
- 🧠 **Conditional Logic**: Scholarships change based on eligibility
- 📋 **User Information Display**: Stores and prints information using arrays
- 🔁 **Menu Navigation**: Allows users to return to the main menu

---

## 🏗️ Program Structure

### Main Program
- **Java Console Application**
- Uses loops, arrays, and conditionals
- Processes user input with `Scanner`

### User Input
The program collects:
- Name
- Minority category
- In-state or Out-of-state status
- School name
- GPA

### Scholarship Matching
Based on GPA:
- **Below 2.50** → No scholarships
- **2.50 – 3.20** → 2 scholarships available
- **Above 3.20** → 3 scholarships available
